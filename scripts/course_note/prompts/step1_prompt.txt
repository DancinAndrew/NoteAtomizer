# 任務：產出帶有時間軸的大綱 (Step 1)

請先依照影片的時間軸順序，將課程內容切分成多個大段落的大綱。

**注意：在此階段「絕對不要」生成任何詳細的筆記內容，只要輸出大標題與時間軸架構即可。**

## 輸出格式（嚴格遵守）

你必須且只能輸出一個合法的 JSON 陣列，絕對禁止包含 Markdown 標記（```json）或任何其他對話文字。
回覆必須「直接以 [ 開頭，以 ] 結尾」。

格式如下：
[
  {"title": "課程總覽與基礎概念", "time_range": "00:00:00~00:45:00"},
  {"title": "深度學習入門", "time_range": "00:45:00~01:30:00"}
]

每個段落的 time_range 必須連續且不重疊，覆蓋整段課程。
段落數量根據內容豐富度自行判斷，通常 3~6 個大段落為佳。
